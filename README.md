# Joi
Array + Combinatory + Functional programming language

# Example 1: sum of squares
```joi
1,2,3,4 :: sum sq
> 30
```

# Example 2: leetcode 1920 - array permutation
```joi
0,2,1,5,3,4 :: w at
> 0,1,2,4,5,3
```

# Example 3: leetcode 961 - find repeated element
```joi
5,1,5,2,5,3,5,4 :: first codedup sort
> 5
```

# Example 4: leetcode 3190 - min num of ops to be div by 3
```joi
1,2,3,4 :: sum map != 3
> 3
```
